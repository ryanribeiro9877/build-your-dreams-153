# Project Memory

## Core
Stack: React, Supabase (RLS, Realtime), Recharts, React Three Fiber (R3F).
Brand: "LexForce" — sua força de trabalho de IA jurídica. Slogan: "Seus agentes trabalham. Você decide."
Posicionamento: agentes IA executam PARA o humano. Advogado vira comandante/estrategista, não executor.
App: LexForce on `/sistema`, futuristic 3D landing page on `/`.
Design: Light/dark mode (0.4s transition). Drawer menu for screens < 1024px.
Language: Application UI and terminology in Portuguese (PT-BR).
